# EchonexVehiclePhysics（增强型车辆物理系统）

基于虚幻引擎蓝图的增强型射线车辆物理系统

## 项目概述

本项目基于 **Ivan Novozhilov 的车辆物理教程** 中的核心概念构建，并加入了额外的增强和自动化功能。目标是创建一个强大且易用的射线车辆系统，减少手动配置的需要。

### 致谢

核心物理方法受到 [Ivan Novozhilov 虚幻引擎车辆教程]的启发。本项目在其方法论基础上进行了扩展和自动化。

## 功能特性

### ✅ 已实现
- **自动车轮半径检测** - 自动计算车轮半径，无需手动测量
- **自动悬挂高度** - 基于车辆配置自动调整悬挂
- 基于射线的悬挂系统
- 自定义质量中心控制
- 纯蓝图实现（无需C++）

### 🚧 开发中
- 前进/后退移动
- 转向机制
- 输入系统
- 更多自动化功能

### 📋 计划中
- 自动调节悬挂刚度
- 漂移物理
- 可定制的操控预设
- 性能优化

## 增强了什么？

**相比基础射线实现：**

| 功能 | 基础方法 | EchonexVehiclePhysics |
|------|---------|----------------------|
| 车轮半径 | 需要手动输入 | ✅ 自动检测 |
| 悬挂高度 | 需要手动调节 | ✅ 自动计算 |
| 设置时间 | ~30分钟 | **~5分钟**（目标） |

"Echonex"增强的重点在于**减少繁琐的手动配置**，同时保持完全的控制力和可定制性。
